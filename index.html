hexo博客部署

发表于 2026-03-12

常用命令

创建目录blog

mkdir blog

初始化hexo

Npx hexo init .

安装依赖

npm install

hexo clean #清理

hexo s #本地预览

hexo g #生成静态文件

hexo d #部署

将本地Hexo博客部署到Github Pages

前提

  • 已安装node.js 和 hexo
  • 本地hexo博客创建
  • Github 已创建仓库，配置完成shh key
  • 配置本地 _config.yml
  • 文章目录
  • 站点概览
