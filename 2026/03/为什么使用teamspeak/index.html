为什么使用teamspeak?

发表于 2026-03-14
字如其名作为团队语音软件。它实现了多人语音交流，还有低占用，低延迟，零广告，非常符合我对软件的使用需求，已用许久十分推荐。

前言

​ 从开始游玩联机游戏时，交流就必不可少了。用过QQ，微信，YY，KOOK，OoPZ等国内语音软件之后，感到不尽人意。

QQ作为最早使用的软件，已经越来越臃肿，多余的功能和后台的高占用是放弃使用的原因。
微信看似简洁的页面，后台占用也不低，最关键的是语音是有延迟的，体感大概1s~2s左右，故也弃用了。
YY印象最深的就是各种弹窗广告了，功能倒是还行，但朋友基本没用了，软件对于我更像老玩家的回忆，故也弃用了。
KOOK，刚使用时，有降噪功能，语音延迟都还不错，但现在后台占用也不低，还有各种商业化的内容，故也弃用了。
Oopz也类似KOOK，商业化的社交频道我用不到，对比teamspeak，还是后者更喜爱些。(以上仅个人感受，接受讨论。)
Mumble低延迟、高质量语音通信的开源软件,未使用不做评价。

​ 所以用过许多的语音软件过后，最终归来还是选择了Teamspeak,字如其名作为团队语音软件。它实现了多人语音交流，还有低占用，低延迟，零广告，非常符合我对软件的使用需求，已用许久十分推荐。

​ 优点讲完，那来说说缺点(假如你能接受就不算是了)，前提是需要一台有公网IP的服务器/云服务器(假如你没有的话，但是学生的话，其实也有免费的方法，可以租用半年的云服务器使用，后续会说明)，和一点点愿意折腾动手实践的能力。

TeamSpeak

配置环境

客户端环境：Windows11
Ts用户端：TS3 Client 64-bit 3.6.2
Ts服务器端：Linux server 64-bit 3.13.7
服务器：阿里云Linux宝塔面板 阿里云专享版 11.1.0

安装

Q：我没有服务器怎么办啊？
A：如果你愿意的话，那就去租一台云服务器，一般各大厂商(阿里云，腾讯云)新人都会有优惠。
假如你是学生，则可以申请学生优惠，免费购买半年的阿里云云服务器使用。https://university.aliyun.com/ 服务器具体配置如图，可以参考。

地域选择离你近的最好，延迟会更低。

购买完成后，等待分配好服务器，进入控制台，选择你的服务器，应用详情。

连接进入宝塔面板使用（开放端口-访问宝塔面板登录）

开放端口
1.找到您的服务器实例。
2.进入 安全组 设置。
3.找到绑定的安全组，点击 配置规则 -> 入站规则。
4.添加一条规则：

 端口范围：填写您想开放的端口（如 8080）。
 授权对象：填写 0.0.0.0/0 (表示允许所有IP访问）
 协议类型：TCP。

5.保存后生效。

需要开放的端口：

端口 协议 说明
9987 UDP 默认语音服务器端口
10011 TCP ServerQuery raw 端口
30033 TCP 文件传输端口
8888 TCP 宝塔面板登录端口

访问宝塔面板登录

根据如图”执行命令”获取面板首页地址，用户名和密码，进入页面登录即可

进入面板后，会提示安装基础环境，按推荐选择LNMP即可。

至此宝塔面板已成功安装。

安装teamspeak服务器

  • 打开宝塔面板左侧栏”终端”，因为默认是root账户，
    为了系统安全，新建用户teamspeak
    useradd teamspeak

  • 切换到/opt目录
    cd /opt

  • 下载并解压Ts服务端安装包
    wget https://files.teamspeak-services.com/releases/server/3.13.7/teamspeak3-server_linux_amd64-3.13.7.tar.bz2 && tar -xjvf teamspeak3-server_linux_amd64-3.13.7.tar.bz2

  • 赋予teamspeak用户权限，并进入该目录切换至teamspeak用户

    chown -R teamspeak:teamspeak /opt/teamspeak3-server_linux_amd64 && cd /opt/teamspeak3-server_linux_amd64 && su teamspeak

  • 同意 TeamSpeak 3 服务器的许可协议

    touch .ts3server_license_accepted

  • 启动服务器(默认使用SQLite搭建)
    ./ts3server_startscript.sh start

    注意复制保存管理员账户和密码还有密钥token！！！

    因为后面使用ts客户端时会用到，建议妥善保存。

  • 到此你已经成功安装teamspeak服务，后续设置自启动等服务可自行搜索配置。接下来进入客户端的使用！

客户端安装与使用

1.下载

https://teamspeak.com/zh-CN/downloads/#ts3client

客户端官网下载地址（根据你的电脑选择对应安装包）

https://dl.tmspk.wiki/https:/github.com/VigorousPro/TS3-Translation_zh-CN/releases/download/snapshot/Chinese_Translation_zh-CN.ts3_translation
汉化包下载地址

2.安装

  • 双击打开安装，下一步同意协议即可。ts支持免登录，所以出现登录时，可以关闭。

  • 提示，先安装完成打开teamspeak后再双击安装汉化包喔。

3.使用

  • 查看你的公网IP地址(在阿里云控制台中可以看到)和上面保存token，我们在这一步需要。

  • 连接-连接到服务器,输入你的公网IP地址，和用户名

  • 因为是第一次进入，所以需要输入上面保存的token。

  • 根据你的需求调节，个性化设置，之后将公网IP分享给你的朋友，然后就可以愉快的使用了。
